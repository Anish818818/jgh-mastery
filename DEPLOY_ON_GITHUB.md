# How to Deploy on GitHub Pages (Free & Public) ğŸŒ

Machan, Netlify eke samahara welawata "Password Protection" auto enable wenawa oyage account settings anuwa. Eka fix karanna amaru nam, **GitHub Pages** use karanna. Eka completely FREE saha PUBLIC.

Menna piyawara (Steps):

### Step 1: GitHub Account Ekak Hadanna
1. Go to [github.com](https://github.com/).
2. "Sign Up" click karala account ekak hadanna (Free).

### Step 2: Aluth Repository Ekak Hadanna
1. Log in unata passe, udinma thiyena `+` icon eka click karala **"New repository"** thoranna.
2. **Repository name:** `jgh-mastery` (ho kamathi namak) denna.
3. **Public** kiyana eka select karanna (Meka Private dammoth site eka wada karanne na).
4. Pahala thiyena **"Create repository"** button eka click karanna.

### Step 3: Files Upload Karanna
1. Dan oya hadapu repository eke **"uploading an existing file"** kiyala link ekak athi. Eka click karanna.
2. Oyage computer eke thiyena `japan ground handling` folder eke thiyena **okkoma files** (index.html, style.css, app.js, images folder etc.) methanata drag & drop karanna.
3. Files upload unata passe, pahala thiyena "Commit changes" button eka click karanna.

### Step 4: Site eka Publish Karanna
1. Repository eke **Settings** tab ekata yanna (Uda menu eke thiyenawa).
2. Wam paththe menu eken **Pages** kiyana eka thoranna.
3. **Branch** kiyana thana `None` kiyala athi. Eka click karala `main` (ho `master`) thoranna.
4. **Save** click karanna.

### Step 5: Link eka Ganna
1. Winadiyak (1 minute) withara indala, ayeth **Settings > Pages** walata yanna.
2. Ethana udimma peewi: **"Your site is live at..."** kiyala link ekak.
3. Anna e link eka thamai oyage Public Website URL eka! ğŸ‰
