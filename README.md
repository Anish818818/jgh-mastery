### Option 1: Netlify Drop (Easiest & Fastest)
*Recommended if you want to see it online in 1 minute.*

1. **Go to Netlify Drop:**  
   Open [app.netlify.com/drop](https://app.netlify.com/drop) in your browser.
2. **Drag & Drop:**  
   Drag your entire `japan ground handling` folder and drop it into the box on that page.
3. **Wait & Visit:**  
   Review eka iwara una gaman (seconds walin), eyala oyata private URL ekak denawa.
   *   Oyata eka customize karanna puluwan (Create account karoth).

### Option 2: GitHub Pages (Permanent & Professional)
*Best if you want to update code frequently.*

1. **Create GitHub Repo:**  
   Go to GitHub and create a new public repository (e.g., `jgh-mastery`).
2. **Upload Files:**  
   Upload all files (`index.html`, `style.css`, `images` folder, etc.) to that repository.
3. **Enable Pages:**  
   Go to **Settings** > **Pages** tab.
   Under "Source", select `main` (or `master`) branch and click Save.
4. **Get Link:**  
   GitHub will give you a link like `yourname.github.io/jgh-mastery`.
